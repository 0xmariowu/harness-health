# agent-lint

Check how well your repo supports AI coding agents. A [Claude Code](https://claude.com/download) plugin.

## Install

```bash
npm install -g @0xmariowu/agent-lint
```

Then start a new Claude Code session and run:

```
/al
```

## What it does

AgentLint scans your repository and scores how well it supports AI coding agents across 5 dimensions: discoverability, context quality, guardrails, workflow integration, and maintainability.

## Requirements

- [Claude Code](https://claude.com/download)
- Node.js 18+
